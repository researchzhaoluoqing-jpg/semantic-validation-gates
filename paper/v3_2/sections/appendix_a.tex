\section{Correction Table: v1 $\to$ v3}
\label{app:correction}

The v1 preprint wrapped the five-gate architecture in differential-geometric
constructions that were not well-posed. The table records each defect and its
replacement. v3.2 adds an empirical footing: the v1 operator set was
instantiated faithfully (Wasserstein distances to reference measures,
EVT/Fr\'echet thresholds, lexicographic $\Pi$) and audited on GSM8K and
TruthfulQA outputs of an instruction-tuned generator ($N=300$). It detected
only $10.6\%$ of incorrect outputs (per-gate AUROC $0.39$--$0.62$; the
\v{C}ech/Hodge logic surrogate below chance at $0.39$), its EVT bounds held
only vacuously (zero rejections, hence zero power), and its classification
flipped on $44\%$ of smallest-quartile perturbations---the v1 construction is
empirically inert as well as theoretically defective. Run artifacts accompany
the released code.
